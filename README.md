## 유니티 기초 공부(똥먹기게임)

### 필요기술 : unity, c#
### tool : unity, visual studio 2022

***

### 게임설명
<pre>
제목 : 똥먹기게임(똥피하기게임)
60초 동안 똥을 먹어서 점수를 올린다.
빨간 똥을 먹을경우 점수가 깍인다.

큰똥  : 3점
중간똥 : 2점
작은똥 : 1점
빨간똥 : -5점
</pre>

### 학습내용
<pre>
- 마우스(손가락)을 클릭하여 캐릭터 움직이기
- 애니메이션 효과
- C# 스크립트 생성
- 장애물 중력 그리고 충돌
- 텍스트 표현 및 시간 표시
- 게임끝내기
</pre>
